# 🚀 How to Upload This to GitHub

Follow these steps to create your GitHub portfolio and get a live website link.

---

## Step 1 — Create a GitHub Account
1. Go to https://github.com
2. Click **Sign Up**
3. Use email: priyapriyankaeluru@gmail.com
4. Choose username: **priyapriyankaeluru** (or any name you prefer)
5. Complete signup and verify your email

---

## Step 2 — Create a New Repository
1. After logging in, click the **+** icon → **New repository**
2. Set:
   - **Repository name:** `qa-portfolio`
   - **Description:** `Manual QA Test Engineer & SAP Functional Tester Portfolio`
   - **Visibility:** ✅ Public (so recruiters can see it)
   - **Do NOT** tick "Add README" (we already have one)
3. Click **Create repository**

---

## Step 3 — Upload All Files

### Option A — Upload via Browser (Easiest)
1. On your new empty repo page, click **uploading an existing file**
2. Drag and drop ALL the files and folders from this zip
3. Add a commit message: `Add QA portfolio files`
4. Click **Commit changes**

> 💡 **Tip:** You need to upload each folder separately.
> - Upload files in `work-samples/` folder first
> - Then `resume-coverletter/` folder
> - Then `docs/` folder
> - Then root files (README.md, index.html, etc.)

### Option B — Using Git (Advanced)
```bash
# In the portfolio-repo folder:
git init
git add .
git commit -m "Initial commit: QA portfolio"
git branch -M main
git remote add origin https://github.com/priyapriyankaeluru/qa-portfolio.git
git push -u origin main
```

---

## Step 4 — Enable GitHub Pages (Live Website)
1. Go to your repo → **Settings** tab
2. Scroll to **Pages** in the left sidebar
3. Under **Source**, select **GitHub Actions**
4. Wait 1-2 minutes
5. Your live portfolio URL will be:
   **https://priyapriyankaeluru.github.io/qa-portfolio**

---

## Step 5 — Add Portfolio Link to LinkedIn
1. Go to your LinkedIn profile
2. Click **Edit profile**
3. Under **Contact info** → **Website** → paste your GitHub Pages URL
4. Also add it to your **About** section
